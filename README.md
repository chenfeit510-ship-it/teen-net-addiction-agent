## 📌 项目概述

**心语伴** 是一个面向青少年的网络健康引导智能体应用。它基于认知行为疗法（CBT）、动机访谈（MI）和积极心理学理论，通过AI对话的方式帮助青少年：

- 🔍 **自我觉察**：认识自己与网络的关系
- 📊 **科学评估**：量化网络依赖程度
- 💡 **认知重构**：识别并调整不合理认知
- 🎯 **行为改变**：制定可执行的改善计划

本项目为 **AI4S 课程大作业**，全程使用 AI 辅助开发（Claude Code / Codex / Workbuddy），符合课程"全程人指导完成"的要求。

## ✨ 功能特性

### 🧠 专业心理对话
- 基于系统提示词工程，AI扮演温暖专业的心理咨询师角色
- 运用认知行为疗法（CBT）、动机访谈（MI）等心理学方法
- 每次回复附带心理分析标签（情绪状态、认知模式、依赖程度、建议方向）

### 📋 网络依赖自测量表
- 8题专业评估量表，覆盖上网时长、情绪依赖、社交影响等维度
- 自动评分并给出健康/轻度/中度/重度四级评估结果
- 可视化进度条展示依赖程度

### 💬 智能对话系统
- 支持多轮上下文对话，记忆完整对话历史
- 6个快捷话题按钮，覆盖常见场景
- 打字动画指示器，提升交互体验

### ⚙️ 灵活配置
- 支持切换 qwen-plus / qwen-turbo / qwen-max 模型
- 可调节对话温度参数（Temperature）
- API Key 本地存储，保护隐私安全
